Google Colab 에서 실행하지 않고 Jupiter Notebook 환경에서 실행하는 방법

- Python 설치
- uv 설치(mac 인 경우 homebrew 추천)
- uv 환경 설정
```
uv init
uv add notebook
uv run jupyter-notebook
```
- 필요 패키지 수동으로 추가
```
uv add numpy
uv add pandas
```
